## Windows Installer (Recommended)

Download the MSI installer for automatic PATH setup:

<Steps>

1. ### Download MSI

   Go to [GitHub Releases](https://github.com/yanlvl99/lune-custom-build/releases/latest) and download `lune-installer.msi`.

2. ### Run Installer

   Double-click the MSI file. Options available:
   - **Install for all users** (requires admin)
   - **Install for current user only**
   - Automatically adds to PATH

3. ### Verify Installation

   Open a new terminal and run:

   ```bash title="Terminal"
   lune --version
   ```

</Steps>

## Manual Installation

### Windows

<Steps>

1. Download `lune-windows-x86_64.zip` from [Releases](https://github.com/yanlvl99/lune-custom-build/releases/latest)

2. Extract `lune.exe` to a folder (e.g., `C:\Program Files\Lune`)

3. Add to PATH:
   - Open Start → "Environment Variables"
   - Edit `Path` → Add the folder path
   - Click OK and restart terminal

</Steps>

### Linux

```bash title="Terminal"
# Download and extract
curl -LO https://github.com/yanlvl99/lune-custom-build/releases/latest/download/lune-linux-x86_64.tar.gz
tar -xzf lune-linux-x86_64.tar.gz

# Move to PATH
sudo mv lune /usr/local/bin/
chmod +x /usr/local/bin/lune

# Verify
lune --version
```

### macOS

```bash title="Terminal"
# Intel Mac
curl -LO https://github.com/yanlvl99/lune-custom-build/releases/latest/download/lune-macos-x86_64.tar.gz

# Apple Silicon (M1/M2/M3)
curl -LO https://github.com/yanlvl99/lune-custom-build/releases/latest/download/lune-macos-aarch64.tar.gz

# Extract and install
tar -xzf lune-macos-*.tar.gz
sudo mv lune /usr/local/bin/
chmod +x /usr/local/bin/lune
```

## Using Rokit

[Rokit](https://github.com/rojo-rbx/rokit) is a toolchain manager for Roblox projects.

```bash title="Terminal"
rokit add yanlvl99/lune-custom-build@0.10.5
```

To upgrade:

```bash title="Terminal"
rokit update lune
```

## Building from Source

Requires [Rust](https://rustup.rs/) installed:

```bash title="Terminal"
git clone https://github.com/yanlvl99/lune-custom-build
cd lune-custom-build
cargo build --release
```

The binary will be at `target/release/lune` (or `lune.exe` on Windows).
